# TechPortfolio - Статический сайт-портфолио

**Версия:** 3.0.0  
**Стек:** HTML5 + Tailwind CSS (CDN) + Vanilla JavaScript + PHP 7.4+ + MySQL

##  Что включено

✅ **Готовая статика** - никаких npm, webpack, сборок  
✅ **5 страниц**: Главная, Услуги, Блог, Обо мне, Контакты  
✅ **Мультиязычность** (RU/EN) - переключатель в шапке  
✅ **Форма связи** с валидацией, hCaptcha и rate limiting  
✅ **MySQL база данных** для хранения контента  
✅ **Простая админ-панель** для управления услугами и заявками  
✅ **Адаптивный дизайн** - работает на всех устройствах  
✅ **SEO-оптимизация** - семантическая разметка, meta-теги  

## 🚀 Установка (5 минут)

### 1. Загрузка файлов на хостинг

**Через FTP (FileZilla):**
1. Подключитесь к хостингу (данные из письма хостинг-провайдера)
2. Откройте папку `public_html` (или `www`)
3. Перетащите ВСЕ файлы из папки `portfolio-static` на сервер
4. Дождитесь завершения загрузки

**Через Файловый менеджер (cPanel/ISPmanager):**
1. Войдите в панель управления хостингом
2. Откройте "Файловый менеджер"
3. Перейдите в `public_html`
4. Нажмите "Загрузить" → выберите все файлы
5. Распакуйте архив (если загружали ZIP)

### 2. Создание базы данных

1. В панели хостинга найдите раздел **"Базы данных MySQL"**
2. Создайте новую базу данных с именем `portfolio_db`
3. Создайте пользователя БД (например, `portfolio_user`) с паролем
4. **ВАЖНО:** Запишите имя базы, пользователя и пароль!
5. Дайте пользователю полные права на базу данных

### 3. Импорт SQL-дампа

1. Откройте **phpMyAdmin** (ссылка в панели хостинга)
2. Выберите созданную базу данных `portfolio_db` в левом меню
3. Перейдите на вкладку **"Импорт"**
4. Нажмите **"Выберите файл"** и выберите `database.sql`
5. Нажмите **"Вперёд"** (Go)
6. Дождитесь сообщения "Импорт успешно завершён"

### 4. Настройка config.php

1. Откройте файл `config.php` в текстовом редакторе
2. Замените данные для подключения к БД:

```php
'db' => [
    'host' => 'localhost',           // обычно localhost
    'name' => 'portfolio_db',        // ваше имя базы
    'user' => 'portfolio_user',      // ваш пользователь БД
    'pass' => 'your_password_here'   // ваш пароль БД
],
